# Feature: WhatsApp Integration

## Overview
Adds a direct communication channel via WhatsApp to the hero section, enabling instant lead contact with predefined messages.

## Technical Details
- **Trigger**: "WhatsApp" button in the hero section navigation.
- **Target**: `https://wa.me/6281222196765`
- **Predefined Message**: "Yo man Happy Lebaran 2026 minal aidzin walfaidzin, mohon maaf lahir batin"
- **Behavior**: Opens in a new browser tab (`target="_blank"`).

## Visual Presentation
- **Background Color**: `#25D366` (WhatsApp Green)
- **Text Color**: `white`
- **Animation**: Inherits `fadeInUp` from the hero button group.
